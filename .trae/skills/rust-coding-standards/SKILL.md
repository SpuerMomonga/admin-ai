本技能提供 Rust 语言特定的编码与工程实践，核心目标是：用类型系统表达约束、用所有权减少共享状态、用错误类型稳定语义、用最小 unsafe 保证安全边界。

## 使用方式

1. 阅读完整规范：references/guidelines.md
2. 设计时先对齐：所有权边界、并发模型、错误类型与公共 API
3. 提交前使用检查清单自检（规范末尾提供）
