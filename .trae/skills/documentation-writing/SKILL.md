本技能提供统一、可复用的文档结构与写作规范，目标是让文档“可执行”：读者能按文档完成搭建、调用、评审与发布。

## 快速开始

1. 选择文档类型并套用模板：README / API / 技术方案 / 变更记录。
2. 写清楚四件事：
   - 谁用（受众）
   - 怎么用（步骤）
   - 约束是什么（前置、权限、兼容性）
   - 出问题怎么查（错误语义、排障入口）
3. 用清单保证完整性（每份模板内置检查项）。
